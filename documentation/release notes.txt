➔ Release notes
Release notes: Yaniv 1.0

Acesso à modelagem
Para abrir a modelagem UML, é necessário utilizar a instalação 17.2 do Visual
Paradigm Community Edition para acessar o arquivo yaniv.vpp,
disponibilizado na pasta documentation.

Como acessar a especificação de requisitos
Para abrir a especificação de requisitos, abra o arquivo Especificacao de Requisitos.pdf.

Execução do código
1. Em um computador com a versão 3.11 do Python instalada, acesse a
pasta src do arquivo descompactado.
2. Execute o arquivo main.py.
2.a. Você pode executar o arquivo pelo terminal com comandos como py
main.py, python main.py ou outros a depender das
configurações da sua instalação.
